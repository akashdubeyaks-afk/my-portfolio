# 🔥 Ultra Premium Portfolio - Awwwards Level

Built with world's best animation & UI CLIs:

## Animation CLI Stack
- **Framer Motion (Motion)** - text reveals, parallax, magnetic buttons
- **Lenis** - ultra smooth scroll
- **GSAP** - advanced animations

## UI CLI Stack  
- **Tailwind CSS CLI** + **shadcn/ui principles**
- Syne + Space Grotesk fonts
- Glassmorphism + huge typography

## Features
- Preloader 0-100%
- Custom Cursor
- Magnetic Buttons
- Conic Gradient Blob
- Parallax Projects
- Infinite Marquee

## Run
```bash
npm install
npm run dev
npm run build
```

## Deploy to GitHub Pages
```bash
npm run build
# push dist to gh-pages branch or use vite plugin
```
